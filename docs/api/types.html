Types and Constants ​

Types ​

Setter<T> ​

一个函数，用于设置动画的属性。

typescript
type Setter<T> = (v: T) => void;

EaseFunc ​

缓动函数。一般定义域为 [0, 1] ，值域为 [0, 1] 。

typescript
type EaseFunc = (x: number) => number;

CanNotAnimateErr ​

无法创建动画时返回，值为 false。

MineHandlerConfig<T> ​

MinePlugin 的 handle 方法所需的配置对象。

属性类型描述
setterSetter<T>设置动画的属性的函数
startT动画开始时的值
endT动画结束时的值
durationnumber动画持续时间，单位为毫秒
easeEaseFunc动画缓动函数

MTimelineConfig ​

属性类型描述
autoStop?boolean时间轴上所有动画都完成后是否自动停止时间轴
onFinish?() => void时间轴上所有动画都完成后的回调函数
onStart?() => void时间轴开始播放的回调函数
driver?MotionDriver时间轴的驱动器

MKeyframe<T extends MineAnimatable> ​

属性类型描述
valuePartial<T>关键帧的数值
duration?number从上一帧的状态到该关键帧的持续时间，单位为毫秒
easing?EaseFunc关键帧的缓动函数

MAnimationConfig ​

属性类型描述
offsetnumber关键帧相对动画开始的偏移量，单位为毫秒
