Plugin ​

MinePlugin ​

MinePlugin 是一个可以通过一定的配置创建出一个 MineHanlder 的对象。通常被用于实现一些较为通用的动画效果。

handle(config: MineHandlerConfig<any>): MineHandler | CanNotAnimateErr ​

尝试通过配置创建出一个 MineHandler。如果创建失败，则会返回 CanNotAnimateErr。

MinePluginManager ​

MinePluginManager 是一个管理器，用于管理 MinePlugin。

register(plugin: MinePlugin) ​

注册一个 MinePlugin。

getHandler(config: MineHandlerConfig<any>): MineHandler | CanNotAnimateErr ​

遍历注册的所有 MinePlugin，尝试通过配置创建出一个 MineHandler。如果创建失败，则会返回 CanNotAnimateErr。
